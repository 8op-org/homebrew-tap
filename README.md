# 8op-org Homebrew Tap

Install gl1tch tools via Homebrew:

```sh
brew tap 8op-org/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `glitch` | gl1tch — AI-native terminal task manager |
| `gl1tch-notify` | gl1tch system tray notification daemon |
| `gl1tch-mud` | gl1tch MUD game world engine |
| `gl1tch-weather` | gl1tch weather widget |

## Installation

```sh
brew tap 8op-org/tap
brew install glitch
brew install gl1tch-notify
brew install gl1tch-mud
brew install gl1tch-weather
```
